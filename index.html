Über dieses Template

Dies ist ein Demo-Template das zeigt, wie man Websites sowohl für Menschen als auch für AI-Agenten optimiert.

Alle Inhalte auf dieser Seite sind Platzhalter und können durch eigene Daten ersetzt werden.

Das Template demonstriert Best Practices für:

  • JSON-LD strukturierte Daten
  • Schema.org Markup
  • Semantisches HTML5
  • AI-Lesbarkeit (LLMs wie ChatGPT, Claude)

Für Menschen

  • Responsives Design – Funktioniert auf allen Geräten
  • Dark/Light Mode – Automatische Anpassung an Systemeinstellungen

  • Performant – Keine externen Dependencies

Für AI-Agenten

  • JSON-LD – Strukturierte Daten im HTML Head
  • Schema.org – Person, Service, FAQPage Schemas
  • Plain-text Mirrors – .txt Dateien für LLM-Tools
  • DataCatalog – Index aller verfügbaren Daten

Technische Details

Dieses Template nutzt eine Dual-Layer-Architektur:

Layer 1: HTML für Menschen

Sauberes, semantisches HTML5 mit Inline-CSS für maximale Performance.

Layer 2: JSON für Maschinen

Strukturierte Daten in JSON-LD und plain-text Mirrors für LLM-Tools.

Warum .txt Mirrors?

LLM-Tools können nur text/plain und text/html lesen.
Daher gibt es für jede .json Datei eine .txt Variante.
