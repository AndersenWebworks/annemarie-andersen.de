Demo Template FAQ — Also available as faq.json

Is this a real profile?

No, this is a demo template. All data are placeholders that should be replaced with your own information.

Can I use this template?

Yes, the template is freely usable. Simply replace all placeholder content with your own data.

What features are included?

JSON-LD structured data, Schema.org markup, semantic HTML5, Dark/Light Mode, Zero JavaScript, WCAG 2.1 compliant, plain-text mirrors for LLM tools.

Where can I find the structured data?

In the HTML head (JSON-LD) and in the /ai/ directory as JSON and TXT files for machine readability.

Do I need to keep the dual-layer architecture?

Yes, for AI readability. The HTML serves humans, while JSON/TXT files serve AI agents and LLMs.
