About This Template

This is a demo template that shows how to optimize websites for both humans and AI agents.

All content on this page consists of placeholders that can be replaced with your own data.

The template demonstrates best practices for:

  • JSON-LD structured data
  • Schema.org markup
  • Semantic HTML5
  • AI readability (LLMs like ChatGPT, Claude)

  • DataCatalog – Index of all available data

Technical Details

This template uses a dual-layer architecture:

Layer 1: HTML for Humans

Clean, semantic HTML5 with inline CSS for maximum performance.

Layer 2: JSON for Machines

Structured data in JSON-LD and plain-text mirrors for LLM tools.

Why .txt mirrors?

LLM tools can only read text/plain and text/html.
Therefore, there's a .txt variant for each .json file.
